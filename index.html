🎯 Farbkombi Regeln


Spielprinzip:

Ein Spieler (Codierer) legt einen geheimen vierstelligen Farbcode aus 6 verfügbaren Farben fest. Der andere Spieler (Rater) versucht diesen Code zu erraten.


Spielablauf:

• Der Rater gibt einen Tipp ab (4 Farbstifte)

• Der Codierer gibt Hinweise:

  - Schwarzer Stift: Richtige Farbe an richtiger Position

  - Weißer Stift: Richtige Farbe an falscher Position

• Die Reihenfolge der Hinweise ist zufällig

• Jede Farbe kann mehrfach verwendet werden


Ziel:

Den Code mit möglichst wenigen Versuchen erraten (maximal 12 Versuche).


Beispiel:

Geheimer Code: Grün-Rot-Blau-Grün

1. Versuch: Rot-Gelb-Rot-Grün → 1 schwarz, 1 weiß

2. Versuch: Grün-Grün-Orange-Rot → 1 schwarz, 2 weiß
